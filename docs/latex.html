\hsize is a TeX primitive that should not be usually used in LaTeX \textwidth is the (constant) width of the total text block \columnwidth is the (constant) width of a single column of text (which is the same as \textwidth for a single column document) \linewidth is a variable that represents the current size of the line of text, whether inside a column or a minipage or a list
